# BitGym #
## What Is BitGym? ##

BitGym is just a few tools designed to help practicing mental conversion between bin, dec, and hex values.

## Included files ##
- **dec2bin.py**: practices translating decimal numbers to binary numbers.
- **subpract.py**: practices subtraction over in the 100-256 span.
